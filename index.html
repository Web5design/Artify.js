What is this?

Inspired by a recent experiment with CSS3 art, and a not so recent experiment with Ascii art. Artify.js is a simple jQuery plugin which allows you to transform images into art with CSS3.

Try it for yourself, your file wont be uploaded to a server.

How does it work?

Artify.js uses HTML5 File API, and CSS3.


View project on Github

How can I use it?

Invoke Artify on a block element to create a drop zone

$('div#drop-zone').artify();

Calling Artify on an image will convert the image in place.

$('img#image-one').artify();

You can provide Artify with two options

$('img#image-one').artify({
	color: 'random', // or hex
	detail: 'medium' // low | high
});
